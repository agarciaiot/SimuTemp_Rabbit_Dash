# Servidor.
Demo, Proyecto con analitica descriptiva. Implementación Álvaro garcía

## Uso, en el powershell
### 1. Construir.
```bash
docker-compose build
```

### 2. Ejecutar.
```bash
docker-compose up -d
```

### 3. Detener.
```bash
docker-compose down
```
